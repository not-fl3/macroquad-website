Macroquad

Macroquad Macroquad is a simple and easy to use game library for Rust programming language.

HTML5

First class browsers support, included WebGL1 and old IOS Safari.

Android

Single command docker-powered android builds.

IOS

Export to XCode project for IOS is supported..

PC

Linux/Mac/Windows are supported with no system dependencies.

Batteries included

UI, efficient 2D rendering, sound system - everything needed to make a 2D game included!
